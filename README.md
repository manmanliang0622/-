## 遊戲簡介

玩家在限時 45 秒內，依照上方**目標色提示**點選對應顏色的貓咪累積分數，點到地雷（狗）或錯誤貓咪則扣分。支援多人連線，遊戲結束後顯示即時排行榜。

---

## 環境需求

| 項目 | 版本 |
|------|------|
| Java JDK | 17 以上（含 17） |
| 作業系統 | Windows（JavaFX Native JAR 已包含於 `./lib`） |
| 網路 | 同一區域網路（LAN）或本機測試 |

> **注意**：不需要另外安裝 JavaFX SDK，所需 JAR 已放在 `./lib` 資料夾中。

## 專案結構

```
cat_catch_javafx/
├── src/catcatch/
│   ├── CatCatchApp.java       主程式入口，場景導航 & 全域設定
│   ├── Theme.java             4 種介面主題（粉色/藍色/綠色/深色）
│   ├── MainMenuScene.java     主選單場景（建立/加入房間）
│   ├── LobbyScene.java        遊戲大廳場景（等待玩家、準備狀態）
│   ├── GameScene.java         遊戲主場景（實體渲染、點擊判斷）
│   ├── ResultScene.java       結果場景（排行榜、再來一局）
│   ├── SettingsScene.java     設定場景（主題、音效、音量）
│   ├── GameServer.java        TCP 遊戲伺服器（支援內嵌模式）
│   ├── GameClient.java        TCP 遊戲用戶端
│   ├── Protocol.java          自訂通訊協定（Pipe 分隔文字 + Base64）
│   ├── ImageProcessor.java    圖片去背處理（BFS Flood-fill + 圓形遮罩）
│   └── SynthAudio.java        程式化音效產生（javax.sound.sampled）
├── assets/
│   ├── cat-gray.jpg           灰色貓咪原圖
│   ├── cat-gold.jpg           金色貓咪原圖
│   ├── cat-snow.jpg           雪白貓咪原圖
│   └── dog.jpg                地雷狗狗原圖
├── lib/                       JavaFX 17 JAR（Windows 版）
├── build_javafx.ps1           編譯腳本
├── run_javafx.ps1             啟動遊戲用戶端
└── run_server.ps1             啟動獨立伺服器
```

## 快速開始

```powershell
# 在專案根目錄執行
.\run_javafx.ps1
```

> 不需要另外跑伺服器！點「建立房間」時，遊戲會自動在背景啟動伺服器。

---

## 詳細操作流程

### 房主（建立房間）

1. 執行 `.\run_javafx.ps1`
2. 輸入暱稱
3. 點選「🏠 建立房間」
   - 程式自動啟動內嵌伺服器並連線
4. 進入遊戲大廳後，畫面顯示：
   - **本機 IP 位址**（分享給同網路的朋友）
   - **4 位數房號**（房間識別碼）
5. 等待所有玩家加入並點選「準備完成」
6. 點選「▶ 開始遊戲」（僅房主可見）

### 其他玩家（加入房間）

1. 執行 `.\run_javafx.ps1`
2. 輸入暱稱
3. 點選「🔗 加入房間」，輸入框展開：
   - **IP 位址**：房主告知的 IP
   - **房號**：4 位數房間碼
4. 點選「🔌 連線加入」
5. 進入大廳後點選「準備完成」

> **同一台電腦測試多人**：IP 填 `127.0.0.1`，開多個 PowerShell 視窗分別執行 `.\run_javafx.ps1`

## 遊戲規則

### 計分方式

| 動作 | 分數變化 |
|------|---------|
| 點到**目標色**貓咪 | **+10 分** |
| 點到其他顏色貓咪 | −5 分 |
| 點到地雷（狗） | −15 分 |

### 遊戲機制

- **遊戲時間**：45 秒
- **目標顏色**：每 5 秒切換一次（上方大提示條顯示）
- **貓咪種類**：灰色貓咪、金色貓咪、雪白貓咪
- **地雷**：狗狗（點到即大扣分）
- **難度遞升**：貓咪數量隨時間增加
- **遊戲結束**：顯示全體排行榜（🥇🥈🥉），可「再來一局」或「返回大廳」

### 目標色提示說明

畫面上方的大型提示區域：
- 顯示目標貓咪的**圓形大頭照**
- 顯示目標顏色名稱（灰色貓咪 / 金色貓咪 / 雪白貓咪）
- 切換目標時背景顏色閃爍提醒
- 目標貓咪的遊戲實體有**發光外框**

## 介面設定

從主選單點選「⚙ 設定」進入設定頁面：

| 選項 | 說明 |
|------|------|
| **主題** | 粉色 / 藍色 / 綠色 / 深色，即時切換 |
| **音效** | 開啟 / 關閉遊戲音效 |
| **音量** | 滑桿調整 0%–100% |

---

## 獨立伺服器模式

若需將伺服器部署在另一台電腦（無頭模式）：

```powershell
# 預設 Port 5050
.\run_server.ps1

# 指定 Port
.\run_server.ps1 -Port 8080
```

其他玩家加入時，IP 填伺服器電腦的 IP，Port 與啟動時指定的一致。

## 技術架構

```
┌─────────────────────────────────────────────────────┐
│                    JavaFX 用戶端                      │
│  MainMenuScene → LobbyScene → GameScene → ResultScene│
│              ↕ GameClient (TCP)                      │
└─────────────────┬───────────────────────────────────┘
                  │ TCP Socket (Port 5050)
┌─────────────────▼───────────────────────────────────┐
│                   GameServer                         │
│  接受連線 → 管理房間 → 廣播遊戲狀態                   │
└─────────────────────────────────────────────────────┘
```

### 核心技術

| 模組 | 技術 |
|------|------|
| 介面框架 | JavaFX 17（Scene Graph、動畫、CSS 樣式） |
| 網路通訊 | Java TCP Socket / ServerSocket |
| 通訊協定 | 自訂 Pipe 分隔文字協定 + Base64 編碼 |
| 圖片處理 | BFS Flood-fill 去背 + 硬圓形遮罩（JavaFX WritableImage） |
| 音效 | javax.sound.sampled 程式化合成音 |
| 主題系統 | Java Enum + 內聯 CSS 樣式字串 |
